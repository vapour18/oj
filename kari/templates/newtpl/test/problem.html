A - 专家来宏福

时间限制: 1000ms 内存限制: 65535KB Special Judge

题目描述

考察完校本部后，专家组就要来宏福校区了。但北京的交通太复杂了，所以从本部到宏福的路线不止一两条。但是专家的时间是宝贵的，尤其不能浪费。

现在，专家车队的队长找到了你，他给了你一张描述了他们待选路线的表（n*n的矩阵A），让你找出行车距离最短的路线。请作为编程高手的你来完成这个艰巨的任务。

表中表示了任意两个路口的连通情况，以及距离。矩阵元素a(i,j)=0表是路口i,j不连通，a(i,j)!=0表示路口i,j的行车距离。其中本部在路口1，宏福校区在路口n。

输入格式

第一行为一个自然数n(1<=n<=100)。

接着n行，每行n个整数，描述待选路线的表(元素的值小于1000)。

输出格式

第一行为一个整数，表示最短行车的距离。

输入样例

4
0 2 3 4
2 0 1 1
3 1 0 0
4 1 0 0

输出样例

3

说明

输入文件巨大，建议使用C/C++的scanf函数。
